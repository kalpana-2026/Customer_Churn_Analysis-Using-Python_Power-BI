# 📊 Customer Churn Analysis

## 🔍 Project Overview
This project analyzes telecom customer data to understand why customers leave (churn) and how to reduce it.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Power BI

## 📈 Key Insights
- Month-to-month contract customers have higher churn
- Customers with low tenure are more likely to leave
- High monthly charges increase churn risk
- High-risk customers identified using business logic

## 📊 Dashboard
Power BI dashboard created to visualize churn patterns and customer behavior.

## 📂 Files Included

- [Python Analysis Notebook](Python_Project.ipynb) → Data analysis  
- [Cleaned Dataset](Cleaned_Churn_data.csv) → Processed dataset  
- [Power BI Dashboard](Python_Project_Dashboard.pbix) → Power BI dashboard
  

## 🚀 Conclusion
This project demonstrates how data analysis and visualization can help businesses reduce customer churn and improve retention strategies.
